Available Features

To Do List

The To Do List allows the user to keep track of their daily tasks. Users create a list of tasks that they can cross-off as they are completed.

Pomodoro Clock

The Pomodoro Clock is a timer that helps users stay focused and motivated. The timer oscillates between 25 minutes of focus time and 5 minutes of break time.
